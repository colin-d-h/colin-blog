### git指令

**合并分支 切换到对应分支后**
```
git merge  xx-xxfeature-YYMMDD
git add. 
git commit -m '--build -d open-account'
git push
```
**创建 切换分支**
```
git branch <newBranchName>
git checkout <branchName>
// 组合
git checkout -b <newBranchName>
```
**删除分支**
```
// 删除本地分支
git branch -d <branchName>
// 查看分支 
//-all 所有分支 
git branch -a
//-remotes 远程分支
git branch -r
// 删除远程分支
git push origin -d <branchName>
```
